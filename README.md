# EE-DataGift-Automation

```
EEDataGift:
  Automation of Gift Data flow on the EE My Account area (windows only!)

Usage:
  EEDataGift [options]

Options:
  --username <username>                          Your EE username (usually your email address)
  --password <password>                          Your EE password
  --donor-telephone <donor-telephone>            e.g. 07700654321
  --recipient-telephone <recipient-telephone>    e.g. 07700123456
  --gift-mb <gift-mb>                            Megabytes to gift (Ensure you have enough!)
  --chrome-path <chrome-path>                    Chrome path e.g. C:\chrome\chrome.exe or .\chrome.exe. Will search PATH if not provided. [default: ]
  --version                                      Show version information
  -?, -h, --help                                 Show help and usage information
```
